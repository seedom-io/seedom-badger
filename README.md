## General dependencies
- NodeJS v8.6.0+

## Seedom dependencies
- seedom-solidity
- seedom-crypter

## Run locally
- `NETWORK=<network> node ./index.js`

## Prepare distribution
- Run `npm run dist`
- Push and deploy to swarm

## Testing
- run locally
- open `https://localhost:3000/badges/<contractAddress>/<participantAddress>.png`
